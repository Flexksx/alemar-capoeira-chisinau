# Alemar Capoeira Chisinau

A simple static website built with Svelte 5 and SvelteKit that provides a library of capoeira songs for our group to practice.

## About

This is a straightforward song reference website for Alemar Capoeira Chisinau. The songs are stored in a JSON file and displayed through a clean, searchable interface to help our members learn and practice capoeira music.

## Tech Stack

- **Svelte 5** - Modern reactive framework
- **SvelteKit** - Static site generation
- **Cloudflare Pages** - Hosting and deployment

## Project Structure

- `webapp/src/lib/data/songs.json` - Library of capoeira songs with lyrics and metadata
- `webapp/src/` - Svelte components and application code

## Development

```bash
cd webapp
pnpm install
pnpm dev
```
